MiniClaw UI Refinement

Design Proposal — Swiss Precision
Philosophy
From Brutalist to Swiss Precision
Keep the monochrome DNA and minimal character. Replace harsh black borders with softer grays. Add subtle border-radius (6-8px) for approachability without losing sharpness. Introduce whisper shadows for depth hierarchy. Every refinement should feel intentional, not decorative.
Token Before After Rationale
--color-black #000000 #111111 Slightly softer than pure black, reduces eye strain
--border 1px solid #000 1px solid #e2e2e2 Borders define structure, not dominate; strong variant for emphasis
border-radius 0px everywhere 4/6/8/12px scale Subtle rounding; 6px default feels precise, not bubbly
shadows none xs/sm/md/lg scale Whisper shadows for depth — dropdowns float, cards lift
gray scale 3 grays 10-step scale Finer contrast control for hierarchy and states
status colors #22c55e / #eab308 / #ef4444 #2a9d5c / #c98a0c / #d44040 Slightly muted for enterprise feel, less saturated
transitions 0.15s ease cubic-bezier(0.4, 0, 0.2, 1) Material-style easing for natural motion
input style bottom-border only bordered container Enclosed input feels more stable and refined
sidebar active full black bg, square black bg + 6px radius Keeps the bold active state, softens edges
message area bg #ffffff #f7f7f7 Subtle tint separates content from chrome
