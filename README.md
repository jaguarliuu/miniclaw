# MiniClaw - AI Agent 实战课程

从零构建生产级 AI Agent 框架的课程代码仓库。

## 项目结构

```
miniclaw/
├── backend/          # 后端代码 (Java/Spring Boot)
├── frontend/         # 前端代码 (Vue 3)
├── docs/             # VitePress 文档站
│   └── chapters/     # 课程章节文档
└── README.md
```

## 快速开始

### 文档预览

```bash
cd docs
npm install
npm run docs:dev
```

### 后端开发

```bash
cd backend
./mvnw spring-boot:run
```

### 前端开发

```bash
cd frontend
npm install
npm run dev
```
