## 课程简介

2026年1月，OpenClaw 以 GitHub 史上最快速度获得 15 万+ Star，催生了 AI Agent 专属社交平台 Moltbook（120万+注册 Agent）。这标志着 AI 从"对话工具"到"行动助手"的范式转变。

MiniClaw 课程采用"先用后学"的教学理念：第1章直接教你使用 OpenClaw 解决实际问题，在开发之前就体验 Agent 的价值；学习过程中用 Agent 辅助学习（代码审查、问题排查）；然后从零手写完整 Agent 系统。

## 你将学到

- ✅ AI Agent 的实际应用场景和使用技巧
- ✅ 五层架构设计方法论
- ✅ ReAct 循环实现原理和工具调用协议设计
- ✅ Memory 工程（真相源 vs 索引、向量检索、Compaction）
- ✅ Skills 系统（解析、选择、工具白名单）
- ✅ 完整的可部署生产级系统（后端+Web UI）

## 适合人群

- 后端开发工程师：具备 Java/Spring Boot 基础，希望转型 AI 应用开发
- AI 应用开发者：会使用 LangChain 等框架，希望理解框架原理
- 架构师/技术 Leader：需要为团队设计 AI Agent 系统架构
- 计算机专业学生：希望通过实战项目提升就业竞争力
