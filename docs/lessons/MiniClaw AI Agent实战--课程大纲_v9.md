# MiniClaw AI Agent 实战课程：从零构建生产级 AI Agent 框架

## 课程基本信息

### 讲师优势
1. 深度理解 OpenClaw 架构设计,掌握 AI Agent 核心原理
2. 从零手写完整 Agent 系统，不依赖任何 AI 框架
3. 工业级实践经验，对标真实生产环境
4. 完整的五层架构设计方法论

### 课程简介
2026年1月，OpenClaw 以 GitHub 史上最快速度获得 15 万+ Star，催生了 AI Agent 专属社交平台 Moltbook（120万+注册 Agent）。这标志着 AI 从"对话工具"到"行动助手"的范式转变。

MiniClaw 课程采用"先用后学"的教学理念：第1章直接教你使用 OpenClaw 解决实际问题，在开发之前就体验 Agent 的价值；学习过程中用 Agent 辅助学习（代码审查、问题排查）；然后从零手写完整 Agent 系统。

课程基于 OpenClaw 工业级架构，采用 Java/Spring Boot 技术栈。不使用任何 AI 框架（LangChain、LlamaIndex 等），让学员深刻理解每一行代码的设计动机。

学完后能够：读懂主流 AI Agent 框架源码、为企业设计 AI Agent 系统、定制化改造现有框架、具备从零设计生产级系统的能力。

### 适合人群
1. 后端开发工程师：具备 Java/Spring Boot 基础，希望转型 AI 应用开发
2. AI 应用开发者：会使用 LangChain 等框架，希望理解框架原理
3. 架构师/技术 Leader：需要为团队设计 AI Agent 系统架构
4. 计算机专业学生：希望通过实战项目提升就业竞争力

### 课程收获
1. 掌握 AI Agent 的实际应用场景和使用技巧
2. 掌握五层架构设计方法论（Control/Execution/Extension/State/Automation）
3. 理解 ReAct 循环实现原理和工具调用协议设计
4. 掌握 Memory 工程（真相源 vs 索引、向量检索、Compaction）
5. 理解 Skills 系统（解析、选择、工具白名单）
6. 获得完整的可部署生产级系统（后端+Web UI+CLI）
7. 具备从零设计 AI Agent 系统的能力

### 课程亮点
1. "先用后学"：开发之前就能体验 Agent 的价值
2. 从零手写，不使用任何 AI Agent 框架
3. 基于 OpenClaw 工业级实践，对标真实生产环境
4. 完整的五层架构，清晰的分层设计
5. 每个 Git Commit 都是可运行的完整版本
6. 深度解析设计动机，不只是"怎么做"，更重要的是"为什么"
7. 支持国产大模型 + Ollama 本地部署
8. 真实应用场景演示（代码审查、日报生成、数据分析等）

### 课程难度
中高级（需要 Java/Spring Boot 基础）

**必须掌握：**
1. Java 基础语法（集合、泛型、Lambda）
